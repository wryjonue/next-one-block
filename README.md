# Next: One Block

A randomized One Block survival experience for Minecraft Bedrock Edition.t
Unlike traditional One Block maps that follow a scripted "Phase" system (e.g., Plains -> Underground -> Desert), **Next: One Block** removes the predictable sequence. 

## Installation
1. Download the `.mcaddon` file.
2. Import it into Minecraft Bedrock.
3. Create a new world Flat > The Void
4. Apply the Behavior.
